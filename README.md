# sonos-display

A simple way to show the current sonos track on an [arduino](https://www.arduino.cc/) + [lcd](https://learn.adafruit.com/adafruit-arduino-lesson-11-lcd-displays-1/overview)

#### setup

`` npm install ``

#### run

`` node index.js ``
